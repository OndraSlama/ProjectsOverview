My (un)finished personal projects
These are some of my personal projects worked on mainly in my free time (sometimes also during school and in work but ssshhh). Some of them aren't sill quite finished and few of them never will be. Clicking on any of them (except those marked as *Still in progress*) will redirect you either to the actual implementation of the project or to site containing more info (such as youtube video, github repository etc.). I hope you will find at least some of them interesting.

Space Tanks

Game for 2 players written in JavaScript using p5.js library.

Controls: Player 1: w-s-a-d-f; Player 2: up-down-left-right-shift.

Table Football

Table football simulated using box2D physics engine. Used for developing strategies for real model of robotic table football model.

Physics Engine

Basic physics Engine built with p5 js library. Supports only circular objects. Later used in Space tank game.

Strategy simulation

Strategy simulation for robotic table footbal written in c++. Will be used for reinforced learning of strategy (next year hopefully :D)

Hexapot

School project aimed at creating six legged moving robot capable of recognizing specific color (red) using Lego Mindstorm.

Automated blinds

Automated sunblinds using esp8266, servo and other modules.

*Still in progress*
